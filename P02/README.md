## Introducció

A la tasca 8 vau triar quina seria la proposta de pàgina web que presentareu davant els responsables de FoodLogístics per substituir la seva pàgina actual. Toca doncs, desplegar aquesta solució per tal que sigui visible pel client.


### Descripció de l’activitat

En primer lloc creeu-vos el repositori corresponent a la tasca web_projecte7, l’enllaç el teniu al Moodle de Projecte Intermodular (P02 web corporativa). Un cop la teniu, cloneu el repositori en local i a dins, completar la informació al README.md i crear una carpeta /docs. A dins copieu el codi corresponent a la pàgina que voleu presentar al client.

Un cop heu confirmat els canvis i els heu pujat a GitHub, configureu el desplegament a GitHub Pages del contingut de docs.

**Cal que tots els membres de l’equip facin aquesta això als seus respectius repositoris.**
